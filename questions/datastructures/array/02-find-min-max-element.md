### Problem statement
<p class="d-tag">easy</p>

Find minimum/maximum element in the array and return its index. If there is more than one min/max element return the first element index

**Input** (min)  
[9, 2, 1, 10, 10, 0, 6, 8, 4, 8] 

**Output**  
5

**Input** (max)  
[9, 2, 1, 10, 10, 0, 6, 8, 4, 8] 

**Output**  
4
