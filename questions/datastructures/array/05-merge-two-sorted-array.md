### Problem statement
<p class="d-tag">easy</p>

Merge two sorted array

Input 1  
[1, 2, 3, 4, 5]  
[4, 5, 6, 7, 8]

Output 1  
[1, 2, 3, 4, 4, 5, 5, 6, 7, 8]

Input 2  
[1, 3, 5, 7, 9, 11, 13]  
[2, 4, 6]


Output 2  
[1, 2, 3, 4, 5, 6, 7, 9, 11, 13]
