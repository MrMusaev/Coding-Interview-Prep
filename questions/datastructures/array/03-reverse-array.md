### Problem statement
<p class="d-tag">easy</p>

Reverse an array of characters 

Additional: Reverse an array in-place, using no extra space

**Input 1**  
["h","e","l","l","o"]

**Output 2**  
["o","l","l","e","h"]

**Input 1**  
["H","a","n","n","a","h"]

**Output 2**  
["h","a","n","n","a","H"]
